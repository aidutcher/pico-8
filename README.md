[PICO-8](https://www.lexaloffle.com/pico-8.php) is a lightweight, limited-feature game engine. 
It includes editors for code, sprites, maps, and music, and is useful for creating small games.

Games are stored in files called "carts," which can be shared or hosted to allow others to play them.  Anyone who has a cart can view the underlying code for it. The root file can  
also include other files by importing them with "#include."

PICO-8 is known for having notable limitations - the sprite resolution, map size, and even amount of code all have limits. The benefit is that this allows developers to ideate and iterate quickly on concepts by removing a lot of the decision-making that is typical in game development.
